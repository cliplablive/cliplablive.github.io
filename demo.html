ClipLab Live TikTok Integration Demo

This page demonstrates the complete end-to-end TikTok Content Posting API flow used by ClipLab Live. The production workflow uses OAuth, creator review, video upload, publish status polling, and publishing history.

End-to-end flow demonstrated

1. Connect TikTok with OAuth

User clicks Connect TikTok and authorizes ClipLab Live. Requested scopes: user.info.basic, video.upload, and video.publish.

2. Select an approved clip

The dashboard shows a generated vertical MP4 with captions, source credit, title, hashtags, and status.

3. Review caption and visibility

The user reviews title, caption, hashtags, and selected TikTok privacy level before upload.

4. Upload through Content Posting API

ClipLab Live initializes the TikTok upload session, sends video chunks, and receives a publish id.

5. Poll publish status

The dashboard polls TikTok status until the upload is complete or returns a retryable error.

6. Store result and cleanup

ClipLab Live stores the publish status and deletes temporary local files after successful publishing.
