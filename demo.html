End-to-end flow demonstrated

1. Connect TikTok with Login Kit

User clicks Connect TikTok and authorizes ClipLab Live. Requested scope: user.info.basic.

2. Read basic account info

ClipLab Live uses user.info.basic only to show the connected TikTok username and confirm the correct account.

3. Select an approved clip

The dashboard shows a generated vertical MP4 with captions, source credit, title, hashtags, and upload status.

4. Review caption and confirm upload

The user reviews title, caption, hashtags, and confirms before TikTok upload starts.

5. Upload through Content Posting API sandbox

ClipLab Live uses video.upload to initialize the sandbox upload session and send the video file.

6. Store result and cleanup

ClipLab Live stores the sandbox upload status and deletes temporary local files after success.
